INCOME (รายรับ)

"จุดเริ่มต้นของทหารโคลน ลงเฉพาะเงินที่ทำให้รวยขึ้นจริง"

  • กฎข้ามเดือน (Month-Ahead): รับเงินเดือนช่วงปลายเดือน ลงเป็น "วันที่ 1 เดือนถัดไป"
  • ห้ามลงเงินทอน, แคชแบ็ก, เงินเพื่อนคืนที่นี่! (ไปลง Journal)
ESBI Target: พยายามเพิ่มสัดส่วนรายได้ช่อง B (ธุรกิจ) และ I (ลงทุน)
TAB 2

EXPENSES (รายจ่าย)

"ทหารโคลนที่ถูกส่งไปรบ จงแน่ใจว่ามันคุ้มค่า"

Vivian Rule (50/30/20):
50-Needs ( fixed cost/จำเป็น )
30-Wants ( lifestyle/ความสุข )
20-Future ( ออม/ลงทุน )
  • The Budget Router (Auto-Sync 1): จ่ายเงินใหม่ซื้อสินทรัพย์ (20-Future) พิมพ์ชื่อ Category ให้ตรงกระเป๋าเงิน ระบบจะหักเงินเดือนและโอนโยกไปกระเป๋าเป้าหมายทันที!
  • จ่ายลดหนี้ด้วยเงินใหม่: ลงทะเบียนชื่อเจ้าหนี้ให้เป๊ะ ระบบจะวิ่งไปลดยอดหนี้คงเหลือให้อัตโนมัติ

    • BALANCE SHEET (งบความมั่งคั่งสะสม)

    TAB 4 / 7

    ASSETS (ทรัพย์สิน)

    สร้าง Cashflow

    "อะไรที่ใส่เงินเข้ากระเป๋าเรา คือทรัพย์สิน"

    • Golden Geese: ลงทะเบียนชื่อทรัพย์สิน (กบข., พอร์ตลงทุน, หุ้น PTT) ตั้งต้นครั้งเดียวที่นี่
    • Auto-Sync 2, 3: ซื้อ/ขายสินทรัพย์ด้วยเงินเก่าใน Journal ระบบจะอัปเดตยอด Current Value ให้อัตโนมัติ
    • Tab 7 (Recv): รับคืนจากลูกหนี้ พิมพ์ชื่อให้ตรง ระบบจะลดยอดลูกหนี้ให้อัตโนมัติ
    TAB 5

    LIABILITIES (หนี้สิน)

    เอาเงินออกจากกระเป๋า

    "หนี้ไม่ใช่เรื่องน่ากลัว หากเราเห็นตัวเลขที่แท้จริง"

    • ชื่อต้องเป๊ะ: ลงทะเบียนชื่อหนี้และดอกเบี้ย (%) ให้ถูกต้อง เพื่อใช้จำลองวันหมดหนี้
    • Auto-Sync 1.5, 4: จ่ายลดหนี้ต้นด้วยเงินเก่า/เงินใหม่ ระบบจะวิ่งมาหักลดยอดหนี้ในนี้ให้อัตโนมัติ
    • ดอกเบี้ยจ่ายถือเป็น Needs ใน Expense ไม่ทำให้หนี้ลด

    Cash Flow Pathway Scenarios

    เข้าใจระบบการไหลของเงินและ Auto-Sync ในแต่ละสถานการณ์จริงที่คุณอาจจะเจอ

    ➔
    SCENARIO A

    Poor Person / Middle Class Pattern

    "Data Path: Income ➔ Envelopes ➔ System Out"

    INCOME ➔ EXPENSE ➔ OUT 💸

    **ผลลัพธ์:** เงินลอยตัว (Tab 6 Buckets) ลดลงเพื่อจ่าย Tabs 2 (Needs/Wants/Giving) โดยไม่มีเงินก้อนไหนวิ่งไปแตะ Tab 4/5 (Assets/Debts)

    SCENARIO B

    Rich Person Pattern (The Wealth Builder)

    "Data Path: Income ➔ Envelopes ➔ Assets ➔ Passive Income"

    INCOME ➔ ASSET ➔

    **ผลลัพธ์:** เงินไหลจาก Tab 1/Tab 3 เข้าไปสร้าง Golden Geese ใน Tab 4 จนเกิด Passive Income (Tab 1 I) ทบกลับมา

    SCENARIO C

    Wealth Builder Mode (Auto-Sync 1)

    "Data Path: Active Income (Tab 2 Future) ➔ System Router ➔ Asset Box (Current Value)"

    INCOME ( Tab 2 )

    -- จ่ายให้ตัวเองก่อน (20%) --

    ASSET BOX ( Tab 4 )

    **ผลลัพธ์:** บันทึก Expense (20-Future) พิมพ์ชื่อ Category ให้ตรงกระเป๋าเงินเดือนใหม่ ระบบจะหักเงินเดือนและโอนโยกไปกระเป๋าเป้าหมายทันที!

    SCENARIO D

    Debt Slayer Mode (Auto-Sync 1.5, 4)

    "Data Path: Active Income (Tab 2) / Old Money (Tab 3) ➔ System Router ➔ Liability Box (Principal)"

    INCOME / ASSET ( Tab 2 / 3 )

    -- จ่ายลดเงินต้น --

    LIABILITY BOX ( Tab 5 )

    **ผลลัพธ์:** บันทึกจ่ายลดเงินต้น พิมพ์ชื่อให้ตรง ระบบจะหักเงินและโยกลดมูลค่าคงเหลือของหนี้ใน Tab 5 ทันที!
